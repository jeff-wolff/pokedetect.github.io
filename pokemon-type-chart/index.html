THERE ARE NO IMMUNITIES IN POKEMON GO

The Pokemon GO Type Chart Calculator lists the traditional Pokemon type chart with resistances (red) and weaknesses (green). There are no immunities in Pokemon GO only resistances.

Venusaur's type chart can be read as: Venusaur resists damage from grass types (0.64x damage multiplier) and fighting, water, electric, and fairy types (0.8x damage multiplier).

Venusaur takes super effective damage (an extra 1.25x damage multiplier) from fire, flying, psychic, and ice types.

There are no double type advantages for Venusaur. A double type advantage (or weakness) adds a 1.56x damage multipler instead of 1.25x.

This Pokemon GO type chart does not account for STAB (Same Type Attack Bonus) which is an extra 1.25x damage multiplier in Pokemon GO.
